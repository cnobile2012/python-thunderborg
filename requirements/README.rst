*********************
Notes on requirements
*********************

If creating a virtual environment for development, the approxeng.txt,
pygame.txt, and monsterweb.txt are included and do not need to be installed
separately.

The freeze.txt file is only used for readthedocs.org, so it must be kept
up-to-date or it may fail building on readthedocs.org.
